DSO3D12 Flasher CLI package

Contents:
- `flash` executable for this operating system (firmware flasher)
- `capture` executable (serial capture tool)
- `LICENSE.txt`
- `README.md`

Usage:
- Run `flash --help` to see firmware flashing commands.
- Run `capture --help` to see serial capture commands.
- USB serial access is required to communicate with the oscilloscope.
